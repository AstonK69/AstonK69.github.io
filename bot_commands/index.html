MatHax Bot Commands!

Avatar: Sends profile picture of any user `(m!avatar )` Ban: Bans a specified user `(m!ban )` Bigrat: Sends bigrat Help: Shows this Kick: Kicks a specified user `(m!kick )` Mcuser: Gives info on given account `(m!mcuser )` Mute: Mutes a user `(m!mute )` Ping: Checks bot is alive Purge: Deletes an amount of messages `(m!purge )` Say: Repeats your message `(m!say )` Saye: Repeats your message in an embed `(m!saye "title" "message")` Setprefix: Allows for the prefix to be changed `(m!setprefix )` Smallrat: Sends smallrat Tempban: Temporarily bans a user `(m!tempban )` Tempmute: Temporarily mutes a user `(m!tempmute )` Unban: Unbans a banned user `(m!unban )` Unmute: Unmutes a muted user `(m!unmute )` Website: Sends mathax website Whois: Sends information on a user `(m!whois )`
